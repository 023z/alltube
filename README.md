alltube
=======

HTML GUI for youtube-dl (http://alltubedownload.net/)

##Setup
The only thing you need to get Alltube working is to download [youtube-dl](https://rg3.github.io/youtube-dl/):

    wget https://yt-dl.org/downloads/latest/youtube-dl
